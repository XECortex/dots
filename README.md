## Stuff I use
### General
  - Window Manager: `i3-gaps`
  - Bar: `polybar`
  - Program Launcher: `rofi`
  - Compositor: `picom-jonaburg`
  
### Appearance:
  - `nitrogen` to set the wallpaper
  - GTK Theme: Matcha Dark Azul
  - Icon Theme: Papirus Dark
  - VSC Theme: GitHub Dark

#### Fonts
